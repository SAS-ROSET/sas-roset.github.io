For more information on GitHub's data collection and privacy practices, please refer to GitHub's Privacy Statement. By using this website, you consent to the data collection practices described above by GitHub.


2. SAS-ROSET Graphical (ROS Encryption Tool) Privacy Policy
The SAS-ROSET Graphical (also referred to as ROS Encryption Tool) does not collect any personal data or user information directly. The software operates locally on your device and does not transmit data to any external servers. We do not collect, store, or share any user data from the use of this software. However, the software relies on the Java Runtime Environment (JRE) to function. The JRE collects a limited set of anonymous technical data (commonly known as telemetry), which is not personally identifiable. This data is collected to help improve the performance and functionality of the JRE. For a detailed description of what data is collected and how it is used, please refer to Oracle's Privacy Policy Please note that this data is not tied to you personally and is used solely for technical purposes to improve the Java platform. We do not integrate any third-party services or tracking mechanisms into the software itself, and the software operates independently without any external dependencies that would collect user information. By downloading and using SAS-ROSET Graphical (ROS Encryption Tool), you acknowledge that while we do not collect personal data, the Java Runtime Environment may collect anonymous technical data as part of its telemetry.
